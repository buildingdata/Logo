### 3.数据团队 Logo

数据团队 Logo 由原版 Logo 改色而来，以节能的绿色为主色调，适用于各类团队图标、开发等场合。例如建筑节能数字小组的图标等。

### 4.源文件

所有矢量 Logo 都是使用 Sketch 进行临摹的，因为源文件好像找不到了，只能拿现有的小尺寸 png 图片描绘绘制而成，部分细节肯定会存在一些细微的差距。源文件在 `/sketch` 目录下。
